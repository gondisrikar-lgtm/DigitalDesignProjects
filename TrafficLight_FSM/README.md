#Traffic Light FSM

A simple Traffic Light Controller implemented using Verilog HDL and Finite State Machine (FSM) concepts.

States

- S0: Road A Green, Road B Red
- S1: Road A Yellow, Road B Red
- S2: Road A Red, Road B Green
- S3: Road A Red, Road B Yellow

Files

- TrafficLight_FSM.v
- TrafficLight_FSM_tb.v

Tools Used

- Verilog HDL
- Xilinx Vivado
